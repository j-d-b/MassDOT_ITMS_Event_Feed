Element laneInfo / Direction
Annotations

DYNAC-based GIS information

The direction for each lane on the start roadway, including both direction of travel for bidirection roadways; e.g. NNN from three northbound lanes and SSSNNN for three southbound lanes and three northbound lanes. The convention is:
- For northbound lanes, the first letter indicates the leftmost travel lane, which is closest to the median;
- For southbound lanes, the first letter indicates the rightmost lane, which is closest to entry and exit ramps;
- For eastbound lanes, the first letter indicates the leftmost travel lane, which is closest to the median;
- For westbound lanes, the first letter indicates the rightmost lane, which is closest to entry and exit ramps.
